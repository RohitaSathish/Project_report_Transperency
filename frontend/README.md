# Group Project Management System - React + Vite

React + Vite frontend for managing group projects with task distribution, progress tracking, peer evaluation, and performance analytics.

## Tech Stack

- **Frontend**: React 18, Vite, React Router, Axios, Recharts
- **Build Tool**: Vite (Fast HMR & Build)

## Installation & Setup

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Frontend Setup

1. Navigate to frontend directory:
```bash
cd frontend
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

Frontend will run on http://localhost:3000

4. Build for production:
```bash
npm run build
```

## Features

### Admin Dashboard
- System-wide statistics
- User management
- Project monitoring

### Faculty Dashboard
- Project statistics with graphs
- Create and manage projects
- Task distribution
- Performance analytics
- Contribution reports
- Mark adjustment

### Student Dashboard
- View assigned tasks
- Update task status
- Submit progress
- Peer evaluation
- Personal contribution reports

## Note

This is the frontend-only version with Vite for faster development. Backend API integration will be added later.
